Como usar o relay Telegram
1
Criar bot e copiar token
Abra o Telegram, procure por @BotFather e envie /newbot. Escolha nome e username — o BotFather retorna um token no formato 123456789:AAF…. Cole o token no passo 1 da card e clique em configurar.
O bot é seu — o token fica salvo criptografado no seu dispositivo, nunca em servidor externo.
2
Vincular conta ao bot
Após configurar o token, o app aguarda no passo 2. Abra o Telegram, encontre seu bot pelo username e envie qualquer mensagem (ex: /start). O app detecta automaticamente seu chat ID.
A outra pessoa precisa fazer os passos 1 e 2 com o mesmo token de bot.
3
Criar grupo relay e configurar ID
No Telegram, crie um grupo e adicione seu bot como administrador com permissão de enviar mensagens. Obtenha o chat ID do grupo — número negativo como -1001234567890 — e cole no passo 3 da card.
Por que o grupo é necessário? O Telegram Bot API só entrega notificações de mensagens enviadas por usuários ao bot. Mensagens do bot para chats pessoais não aparecem no polling. Num grupo onde o bot é admin, todos os posts chegam via getUpdates — por isso o download automático funciona apenas com o grupo configurado.

Dica para obter o ID: encaminhe uma mensagem do grupo para @userinfobot. Ambos os peers usam o mesmo grupo; o QR code já inclui o ID automaticamente.
4
Conectar e usar
Na seção conectar, preencha o ID do peer e o chat ID do peer. Clique em conectar — o app tenta P2P primeiro e muda automaticamente para o relay Telegram se o P2P falhar (NAT bloqueado, ICE failure). Você também pode ativar o relay manualmente com via telegram.
Escaneie o QR code do peer para preencher ID e chat ID automaticamente.
Os arquivos são criptografados? Sim. Todos os dados são criptografados com AES-256-GCM antes de sair do seu dispositivo — o bot e o Telegram veem apenas bytes aleatórios.
Limite de tamanho via Telegram? 50 MB por arquivo. Arquivos maiores só podem ser transferidos via P2P.
