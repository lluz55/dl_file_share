Como usar o relay Telegram
1
Crie um bot no Telegram
Abra o Telegram e procure por @BotFather. Envie /newbot, escolha um nome e username para o bot. Ao final, o BotFather enviará um token no formato 123456789:AAF….
O bot é seu — nenhum servidor externo tem acesso ao token.
2
Configure o token aqui
Cole o token no campo "Token do bot" e clique em configurar. O app valida o token e salva criptografado — ele nunca fica visível no armazenamento.
3
Vincule sua conta ao bot
Após configurar, abra o Telegram, encontre seu bot pelo username e envie qualquer mensagem (ex: /start). O app detecta automaticamente seu chat ID — o número que identifica você nesse bot.
4
Compartilhe seu chat ID (e o do peer)
Seu chat ID aparece na card após o passo 3. A outra pessoa precisa fazer os mesmos passos 1–3 com o mesmo token de bot e te enviar o chat ID dela.
Se você usar o QR code para convidar alguém, o chat ID já vem preenchido automaticamente na página do peer.
5
Conecte com fallback automático
Na seção conectar, preencha o ID do peer e o Telegram chat ID do peer. Clique em conectar normalmente — o app tenta P2P primeiro. Se o P2P falhar (ICE failure, NAT bloqueado), muda automaticamente para o relay Telegram.
Você também pode ativar o relay manualmente a qualquer momento clicando em via telegram no formulário de conexão.
Os arquivos são criptografados? Sim. Todos os dados são criptografados com AES-256-GCM antes de sair do seu dispositivo — o bot e o Telegram veem apenas bytes aleatórios.
Limite de tamanho via Telegram? 50 MB por arquivo. Arquivos maiores só podem ser transferidos via P2P.
